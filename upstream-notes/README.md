# Upstream Contributions

This directory tracks bugs, improvements, and contributions we want to give back to the open-source projects Caz depends on.

## Format

For each finding, create a file named: `{project}-{short-description}.md`

Example: `ollama-python-streaming-timeout.md`

Include:
- **Project:** Name and repo URL
- **Type:** Bug / Feature / Documentation / Performance
- **Description:** What we found
- **How we found it:** Context from building Caz
- **Status:** Identified / Issue Filed / PR Submitted / Merged
- **Links:** Issue or PR URLs
